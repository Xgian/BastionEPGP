    Revision 234:
    
    - Added mining nodes data file
    
    - API change: added Tourist:GetMiningNode(nodeObjectID)
    Returns mining node data containing mining node name, node object ID, ore name, ore item ID, minimum skill level and a collection of zone mapIDs with the number of nodes, for each zone in which the node can be found.
    
    - API change: added Tourist:GetMiningSkillColor(nodeObjectID, currentSkill)
    Returns an r, g and b value representing the gathering difficulty of a specific mining node, given the current player skill
    
    - API change: added Tourist:IterateMiningNodes()
    Returns mining node data as described above for GetMiningNode, for each known standard mining node
    
    - API change: added Tourist:IterateMiningNodesByZone(mapID)
    Returns mining node data containing mining node name, node object ID, ore name, ore item ID, minimum skill level and the number of nodes in the zone, for each standard mining node that can be found in the zone specified by mapID
    
    - API change: added Tourist:IterateZonesByMiningNode(miningNodeObjectID)
    Returns the mapIDs of the zones in which the specified mining node can be found
